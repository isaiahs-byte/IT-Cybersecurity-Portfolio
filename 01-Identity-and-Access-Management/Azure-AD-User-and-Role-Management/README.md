# Azure AD User and Role Management Lab

## Objective
This lab demonstrates user account creation, role assignment, authentication management, and Microsoft Entra ID administrative configuration inside Microsoft Azure.

## Skills Practiced
- Microsoft Entra ID administration
- User account management
- Administrative role assignment
- Authentication method configuration
- Conditional Access navigation
- Azure identity management
- Cloud security fundamentals

## Lab Tasks Completed
- Created Azure lab user
- Reviewed sign-in logs
- Configured authentication methods
- Assigned Directory Reader role
- Navigated Conditional Access
- Reviewed administrative units
- Explored enterprise applications

## Tools Used
- Microsoft Azure
- Microsoft Entra ID
- GitHub
